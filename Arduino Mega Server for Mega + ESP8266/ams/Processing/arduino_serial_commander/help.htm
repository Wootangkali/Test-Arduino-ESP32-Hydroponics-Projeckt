Встроенный язык программирования

Макросы

% (0x25) — базовый префикс макросов

& (0x26) — подключение ссылок _one*.htm

# (0x23) — вставка всех *.js файлов

1 (0x31) — вставка хедера _header.htm

2 (0x32) — вставка dash-панели _dash.htm

3 (0x33) — вставка меню _menu.htm

4 (0x34) — вставка футера _footer.htm

6 (0x36) — вставка плиточного меню _alt.htm

7 (0x37) — вставка плиточного CSS style_a*.css

! (0x21) — вставка IP адреса

? (0x3F) — вставка библиотеки Flotr2

+ (0x2B) — вставка библиотеки Processing

@ (0x40) — вставка библиотеки Three

= (0x3D) — вставка библиотеки jQuery

Пояснения

Макрос состоит из двух символов — базового префикса ( % ) и идентификатора (без пробела между ними). Идентификатор определяет операцию, которую нужно произвести в том месте файла, где он встречается. Это может быть вставка IP адреса системы, вставка кода библиотеки или любое другое заранее определённое действие.

Здесь представлены макросы, которые уже имеются в системе. Вы можете дополнить этот список любыми необходимыми вам действиями по вставке информации в передаваемые сервером файлы.
