Лицензия

Arduino Mega Server. Свободное программное обеспечение только для личного использования. Поставляется «как есть» и без каких-либо гарантий. Права на сторонние библиотеки принадлежат их владельцам.

Коммерческое использование. Запрещается использование этого программного обеспечения, а также его частей и производных от него, в коммерческих целях, на производстве, в составе тиражируемых проектов и для любого, прямого или косвенного, извлечения выгоды. Для коммерческого использования и проектов существует специально для этого предназначенная версия AMS Pro с соответствующим лицензированием.


Версии

Изменения в 0.17 версии

Главное изменение в 0.17 версии Arduino Mega Server — это переход с устаревшей базовой версии Arduino IDE 1.6.5 на новую версию среды разработки 1.8.5. Кроме этого, код системы значительно доработан и улучшен, устранены найденные ошибки и недочёты.
